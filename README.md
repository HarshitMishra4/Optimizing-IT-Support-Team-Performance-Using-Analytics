Title of the Project

Customer Support Ticket Optimization

Problem Statement

Organizations receive a large number of customer support tickets daily, making it difficult to track performance, response time, and customer satisfaction efficiently.

Dataset Description

The dataset contains records of customer support tickets with key attributes such as:

Ticket ID – Unique identifier for each ticket
Customer Name/ID – Customer details
Issue Type / Category – Type of problem reported
Priority Level – Low, Medium, High, Critical
Status – Open, In Progress, Closed, Resolved
Created Date – When the ticket was raised
Resolved Date – When the issue was resolved
Response Time – Time taken to respond
Resolution Time – Time taken to close the ticket
Agent Name – Support staff handling the ticket
Customer Satisfaction Score (CSAT) – Feedback rating

KPI

The following Dataset is based on a single KPI i.e Total Revenue generated.

Dashboard Description

The Dashboard consists of 4 visual representation like scatter chart, bar graphs (horizontal and vertical) and a pie chart with a slicer based on total revenue  based on resolution time (in hrs) and so  the values in the visuals are adjusted accordingly. The Title of the dashboard is Customer Support Ticket Optimization. The KPI used here is Total revenue which was calculated using DAX and formulated as Customer Satisfaction Score * Resolution Time. 

Key Insights



Recommendations

Enhance communication to improve CSAT scores.
Improve response time for high-priority tickets.
 
Tools Used

Power BI – Dashboard creation and visualization.
DAX (Data Analysis Expressions) – KPI calculations and measures.
Power Query editor – Data cleaning and transformation.
xls file – Dataset source.
